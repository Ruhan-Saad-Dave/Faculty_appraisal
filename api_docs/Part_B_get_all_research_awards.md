# Retrieve All Research Awards

**Endpoint Name:** Retrieve All Research Awards  
**URL Path:** `/api/v1/part-b/research-awards`  
**Method:** `GET`

## Description
Retrieves all research award records in the system.

## Access Control
- **Authorized Roles:** Admin, Dean, VC.

## Request Data
- **Query Parameters:** `skip`, `limit`.

## Response Data
- **Code:** `200 OK`
