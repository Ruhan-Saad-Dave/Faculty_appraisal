# Retrieve All Industrial Training

**Endpoint Name:** Retrieve All Industrial Training  
**URL Path:** `/api/v1/part-b/industrial-training`  
**Method:** `GET`

## Description
Retrieves all industrial training records across the institution.

## Access Control
- **Authorized Roles:** Admin, Dean, VC.

## Request Data
- **Query Parameters:** `skip`, `limit`.

## Response Data
- **Code:** `200 OK`
