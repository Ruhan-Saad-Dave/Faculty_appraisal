# Retrieve All Conference Papers

**Endpoint Name:** Retrieve All Conference Papers  
**URL Path:** `/api/v1/part-b/conferences`  
**Method:** `GET`

## Description
Retrieves all conference paper records across the institution.

## Access Control
- **Authorized Roles:** Admin, Dean, VC.

## Request Data
- **Query Parameters:**
  - `skip` (int), `limit` (int).

## Response Data
- **Code:** `200 OK`
