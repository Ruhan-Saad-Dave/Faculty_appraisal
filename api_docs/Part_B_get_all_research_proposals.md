# Retrieve All Research Proposals

**Endpoint Name:** Retrieve All Research Proposals  
**URL Path:** `/api/v1/part-b/research-proposals`  
**Method:** `GET`

## Description

- **Query Parameters:** `skip`, `limit`.

## Response Data
