# Retrieve All ICT Pedagogy

**Endpoint Name:** Retrieve All ICT Pedagogy  
**URL Path:** `/api/v1/part-b/pedagogy`  
**Method:** `GET`

## Description
Retrieves all ICT pedagogy records across the institution.

## Access Control
- **Authorized Roles:** Admin, Dean, VC.

## Request Data
- **Query Parameters:** `skip`, `limit`.

## Response Data
- **Code:** `200 OK`
