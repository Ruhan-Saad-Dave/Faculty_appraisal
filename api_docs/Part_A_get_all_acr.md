# Get All ACR Entries

**URL Path:** `/api/v1/part-a/acr`
**HTTP Method:** `GET`
**Description:** Retrieves all ACR entries in the system. Restricted to high-level roles.

## Request Data
- **Type:** N/A

## Response Data
- **Success Status Code:** 200 OK
- **Fields:** List of ACR objects.

## Access Control
- **Roles:** `admin`, `dean`, `vc`.
