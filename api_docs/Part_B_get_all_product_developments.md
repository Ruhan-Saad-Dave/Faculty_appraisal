# Retrieve All Product Developments

**Endpoint Name:** Retrieve All Product Developments  
**URL Path:** `/api/v1/part-b/products`  
**Method:** `GET`

## Description
Retrieves all product development records across the institution.

## Access Control
- **Authorized Roles:** Admin, Dean, VC.

## Request Data
- **Query Parameters:** `skip`, `limit`.

## Response Data
- **Code:** `200 OK`
